# Test Project
## Использовались:
- React
- Tailwind CSS / SCSS
- Effector
- Effector Forms
- React Calendar

## Как запустить:
- Склонировать репозиторий
- `yarn`
- `yarn start`

## Выполненные требования:
### Календарь
- Даты, на которые назначены события, подсвечиваются
### Список событий:
- Список
- Форма создание события с валидацией ошибок
- Предварительная форма изменения события
